# 📚 Library Management System (C)

A console-based Library Management System developed using the C programming language and fundamental data structures.

## 🚀 Features
- Add new books
- Display all books
- Issue books
- Return books
- Track book availability

## 🛠️ Technologies Used
- C Programming
- Arrays
- Structures

## 📖 Description
This project implements a basic library system using arrays and structures to manage book records. It allows users to perform essential operations like adding, issuing, and returning books.

The system demonstrates the practical use of fundamental programming concepts in C.

## 🎯 Learning Outcomes
- Understanding of structures and arrays
- Implementation of menu-driven programs
- Real-world problem solving using C

## ▶️ How to Run
1. Compile:
   gcc library.c -o library
2. Run:
   ./library
